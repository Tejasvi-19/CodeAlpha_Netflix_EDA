# Netflix Data Analysis - CodeAlpha Internship

## Project Overview
This project performs Exploratory Data Analysis (EDA) on Netflix dataset using Python.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Tasks Performed
- Data Cleaning
- Handling Missing Values
- Movies vs TV Shows Analysis
- Country-wise Analysis
- Ratings Analysis
- Release Year Analysis
- Data Visualization

## Key Insights
1. Netflix contains more Movies than TV Shows.
2. United States has the highest content.
3. TV-MA is the most common rating.
4. Netflix content increased rapidly after 2015.

## Files Included
- eda.py
- netflix_titles.csv
- Graph screenshots

## Outcome
Successfully analyzed Netflix data and generated insights using EDA techniques.
